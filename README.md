# mouth_cut_out

## 概要
HSV色空間のS成分・V成分を用いて、
クリックした画素と類似した色領域を二値化し、
輪郭抽出によって対象領域を矩形で切り出す画像処理ツールです。

## 使用技術
- Python
- OpenCV
- NumPy

## 機能
- マウスクリックによる色指定
- 類似色領域の二値化
- モルフォロジー処理によるノイズ除去
- 輪郭抽出と矩形表示
- ダブルクリックによる領域切り出し・保存

## 操作方法
1. original画像をクリックすると二値化画像が生成される
2. binary画像の白領域を同じ場所で2回クリックすると切り出し画像を保存

## 注意
本プログラムはOpenCVのGUI機能を使用しているため、
ローカル環境での実行を想定しています。
